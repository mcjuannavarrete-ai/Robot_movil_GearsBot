## Tarea 5
## Descripción
En esta práctica se programó un robot LEGO EV3 para avanzar lo más recto posible en la arena Gyro Challenges - Straight Run de GearsBot.
El robot utiliza un sensor giroscópico para medir continuamente su orientación. A partir de esa lectura, el programa calcula el error respecto al ángulo objetivo y ajusta de forma proporcional la velocidad de los motores izquierdo y derecho.
El propósito es corregir desviaciones durante el movimiento para evitar que el robot se desplace hacia un lado o se salga de la arena.

## Hardware utilizado
* LEGO Mindstorms EV3
* EV3 Brick
* 2 Motores Grandes EV3
* Puerto A  Motor izquierdo
* Puerto B  Motor derecho
* Sensor ultrasónico EV3.

## Software utilizado
* Pybricks MicroPython
* Visual Studio Code
