## Tarea 4
## Descripción
En esta práctica se programó un robot LEGO EV3 para seguir una línea utilizando un sensor de color, se dividió en dos partes, una usando ON/OFF y la segunda parte un algoritmo de control proporcional.
El sensor mide la cantidad de luz reflejada por la superficie. Con esa lectura, el programa calcula qué tan lejos se encuentra el robot del valor objetivo y modifica la velocidad de cada motor para corregir la trayectoria.
A diferencia del control ON/OFF, el control proporcional no aplica siempre el mismo giro. La corrección aumenta o disminuye según el error detectado

## Hardware utilizado
* LEGO Mindstorms EV3
* EV3 Brick
* 2 Motores Grandes EV3
* Puerto A  Motor izquierdo
* Puerto B  Motor derecho
* Sensor ultrasónico EV3.
