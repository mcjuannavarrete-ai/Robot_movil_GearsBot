## Tarea 2
## Descripción
Esta actividad consiste en programar un robot LEGO EV3 para realizar un recorrido mediante motores diferenciales. El robot avanza en varios tramos rectos y efectúa cuatro giros a la derecha, con el propósito de aproximarse a una trayectoria cerrada, similar a un cuadrado

* El programa está desarrollado en Pybricks MicroPython y controla dos motores conectados al bloque EV3.

## Objetivo
Programar el robot EV3 para:
* Configurar los motores izquierdo y derecho.
* Avanzar durante intervalos de tiempo definidos.
* Girar a la derecha sobre su propio eje.
* Ajustar velocidades y tiempos para corregir la trayectoria.
* Realizar cuatro cambios de dirección consecutivos.
* Detener el robot al finalizar el recorrido.

## Hardware utilizado
* LEGO Mindstorms EV3
* EV3 Brick
* 2 Motores Grandes EV3
* Puerto A  Motor izquierdo
* Puerto B  Motor derecho

## Software utilizado
* Pybricks MicroPython
* Visual Studio Code
