## Tarea 3
## Descripción
En esta práctica se programó un robot LEGO EV3 con Pybricks MicroPython para desplazarse de forma autónoma y evitar obstáculos utilizando un sensor ultrasónico.

El robot avanza mientras la distancia al obstáculo sea mayor a la distancia mínima establecida. Cuando el sensor detecta un objeto cercano, el robot ejecuta una rutina de evasión: se detiene, retrocede, gira a la derecha y avanza para continuar su trayectoria.

El programa aplica un sistema de control en lazo cerrado, ya que las decisiones de movimiento se toman a partir de la información obtenida por el sensor ultrasónico.
Actividades para programar LEGO EV3 con python usando GearsBot

#Objetivo
Desarrollar un programa para un robot móvil EV3 que mida la distancia a un obstáculo y modifique su trayectoria mediante una rutina de evasión.
